# Data Arena — Mandatory Article Requirements

> **Publish gate:** An article must **not** be submitted or published unless every requirement below is clearly present in the article.

See the live version in the app under **Write → Writing standards → Mandatory standards** for the full, always up-to-date document.

## Quick gate (8 required sections)

1. **What is it?** — plain definition  
2. **Why does it matter?** — problem and motivation  
3. **When to use it** — use cases + when **not** to use it  
4. **How it works** — internal flow / mechanics  
5. **How to use it** — practical steps or examples  
6. **Real-world usage** — production or interview relevance  
7. **Common mistakes** — pitfalls and misconceptions  
8. **Key takeaways** — short summary at the end  

## Golden rule

Every article must answer: What? Why? How does it work? When to use? Real-world usage?

# Data Arena — Essential Writing Guidelines

> **Purpose:** How to write on Data Arena — tone, structure, formatting, and clarity.

See the live version in the app under **Write → Writing standards → Essential guidelines** for the full document.

## Core principles

- Write for the reader, not to show off
- One idea per section
- Explain before you show code
- Simple → advanced progression
- Short paragraphs, clear headings
- Define every term on first use
- Every code block needs purpose + explanation
- Professional, accurate, self-contained articles
